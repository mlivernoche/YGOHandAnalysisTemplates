# YGOHandAnalysisTemplates

This is a repository for .NET templates. These are useful for starting new console-based analyzers relying upon [YGOHandAnalysisFramework](https://github.com/mlivernoche/YGOHandAnalysisFramework) and [CardSourceGenerator](https://github.com/mlivernoche/CardSourceGenerator).

## Related Repositories
1. [CardSourceGenerator](https://github.com/mlivernoche/CardSourceGenerator)
2. [YGOHandAnalysisFramework](https://github.com/mlivernoche/YGOHandAnalysisFramework)
3. [AdvancedDeckBuilder](https://github.com/mlivernoche/AdvancedDeckBuilder)
4. [YGOHandAnalysisTemplates](https://github.com/mlivernoche/YGOHandAnalysisTemplates)

## How To Install

[This repository is available as a NuGet package.](https://www.nuget.org/packages/YGOHandAnalysisTemplates)
